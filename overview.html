CitePal - Overview

Motivation:

The idea behind the CitePaL is to provide build a tool that aids researchers in quickly accessing relevant background and motivation for a given paper. The main use case is a researcher who encounters a new paper in a line of research that they are less familiar with. Using our tool, the researcher would input a single paper, and receive a list of three or four key papers that the given paper builds on, along with an interactive interface to explore the corpus of reference papers.

Reference Hierarchy:

Information on the reference hierarchy. I group papers using hierachical clustering based on the SPECTER embeddings provided by Semantic Scholar. Then, to interpret the groups, I use GPT3 to generate a title for each group based on the titles of the groups below it. The group titles are the key aspect of the demo and allow the user to browse an interpretable hieararchy.
